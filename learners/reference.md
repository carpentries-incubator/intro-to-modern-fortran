## Glossary

[Compiled Language]{#compiled-language}
:   Code written in a compiled programming language needs to be
translated into machine code before it can be executed.
This translation is done by a compiler.
Examples of compiled languages include Fortran, C, and C++.
See [compiled vs interpreted languages](https://www.freecodecamp.org/news/compiled-versus-interpreted-languages/)
for more details.
